# 🛒 BuyWorld – Frontend E-Commerce Website

BuyWorld is a responsive and interactive **Frontend E-Commerce Website** developed using **HTML5, CSS3, and JavaScript**. The project simulates a real-world online shopping platform where users can register, log in, browse products, manage their shopping cart, complete the checkout process, and view their profile and order history.

The application uses **Local Storage** to store user information, shopping cart data, and order details, providing a complete shopping experience without requiring a backend database.

---

## 📖 Project Overview

BuyWorld is designed to demonstrate modern frontend web development concepts by creating a fully functional e-commerce website. It provides an intuitive and user-friendly interface that allows users to shop for products, manage their accounts, and place orders efficiently.

## 🚀 Installation and Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/BuyWorld.git
```

### 2. Open the Project

Open the project folder using **Visual Studio Code**.

### 3. Install Live Server

Install the **Live Server** extension from the VS Code Extensions Marketplace.

### 4. Run the Project

Open `index.html`, right-click, and select **Open with Live Server**.

The application will launch in your default web browser.

## 🛍️ Application Workflow

1. Register a new user account.
2. Login using your registered credentials.
3. Browse the available products.
4. Search, filter, and sort products.
5. View detailed product information.
6. Add products to the shopping cart.
7. Update cart quantities or remove items.
8. Proceed to the checkout page.
9. Enter shipping information.
10. Select a payment method.
11. Place the order successfully.
12. View profile information and order history.
